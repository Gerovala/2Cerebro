Possuem retornos [[Dados#Bool|booleanos]], porem só utiliza valores são verdadeiros, se sim ele retorna verdadeiro casso o contrario é falso

- &&/and - Ele checa se todos os valores são verdadeiros, se sim ele retorna verdadeiro
- ||/or - ele checa se algum valor é verdadeiro, se sim ele retorna verdadeiro
- Not - ele retorna True se o valor for False
