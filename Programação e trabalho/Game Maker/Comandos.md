## Randomizer 
O computador nunca cria valores aleatórios
ele cria uma lista de valores que são especificados pela semente (seed)
No game maker o codigo randomize serve para trocar a semente e codigos como *close*,Random e ranom_range: Irão criar uma lista de valores a sua preferencia
